# AI E-commerce Manager (Multi-Agent System)

This project demonstrates a multi-agent AI system built using CrewAI to analyze e-commerce data and generate business insights.

## Features
- Sales analysis
- Inventory risk detection
- Marketing strategy generation
- Automated PDF reports

## Tech Stack
- Python
- CrewAI
- Pandas
- ReportLab

## Dataset
Kaggle Brazilian E-commerce dataset (100k+ orders)

```bash
python main.py
